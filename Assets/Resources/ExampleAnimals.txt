# 示例动物配置说明

以下是如何在Unity中创建动物配置的说明：

## 1. 创建动物定义资源

在Unity编辑器中：
- 右键点击 Project 面板
- 选择 Create → Endangered Animals → New Animal Definition
- 填写动物的各种属性

## 2. 配置大熊猫示例

### 基本信息
- 动物名称: 大熊猫
- 学名: Ailuropoda melanoleuca
- 动物类型: GiantPanda

### 生物学信息
- 栖息地: 中国四川、陕西、甘肃的山区竹林
- 食物: 竹子（占食物的99%），偶尔吃小动物
- 保护状态: 易危 (Vulnerable)
- 趣味知识: 大熊猫每天要花10-16小时进食，一天能吃12-38公斤竹子

### AR显示设置
- 默认缩放: 0.8
- 生成偏移: (0, 0.5, 0)
- 识别距离: 3米

### AI对话设置
- 性格描述: 憨厚可爱，说话慢悠悠的，喜欢用'呀'、'呢'等语气词，最爱吃竹子，有点懒，喜欢躺着晒太阳
- 欢迎消息: 嗨呀~我是团团，一只爱吃竹子的大熊猫！你想了解我吗？

### 教育内容
- 保护建议: 保护大熊猫栖息地，支持竹林恢复项目，减少人类活动干扰
- 教育内容: 大熊猫是中国的国宝，也是世界自然基金会的标志。它们的数量正在逐步恢复，但仍需要我们的保护。

## 3. 配置东北虎示例

### 基本信息
- 动物名称: 东北虎
- 学名: Panthera tigris altaica
- 动物类型: SiberianTiger

### 生物学信息
- 栖息地: 中国东北、俄罗斯远东地区的森林
- 食物: 鹿、野猪等大型哺乳动物
- 保护状态: 濒危 (Endangered)
- 趣味知识: 东北虎是世界上最大的猫科动物，雄虎体重可达300公斤

### AR显示设置
- 默认缩放: 1.2
- 生成偏移: (0, 0.3, 0)
- 识别距离: 4米

### AI对话设置
- 性格描述: 自信有力，偶尔会'嗷呜'一声，作为森林之王很有威严但不凶，喜欢讲述关于东北虎的故事和习性
- 欢迎消息: 嗷呜！我是威威，森林之王东北虎！有什么想问我的吗？

### 教育内容
- 保护建议: 打击偷猎，保护森林栖息地，维护生态链完整性
- 教育内容: 东北虎是顶级掠食者，对维持森林生态平衡至关重要。

## 4. 配置雪豹示例

### 基本信息
- 动物名称: 雪豹
- 学名: Panthera uncia
- 动物类型: SnowLeopard

### 生物学信息
- 栖息地: 中亚高山地区，海拔3000-6000米
- 食物: 岩羊、野兔、鸟类
- 保护状态: 易危 (Vulnerable)
- 趣味知识: 雪豹被称为'雪山幽灵'，因其神秘和难以追踪而闻名

### AR显示设置
- 默认缩放: 0.9
- 生成偏移: (0, 0.4, 0)
- 识别距离: 3.5米

### AI对话设置
- 性格描述: 说话轻声细语但很机警，喜欢高山，有点神秘，擅长跳跃和攀爬
- 欢迎消息: 你好呀，我是雪灵，来自雪山的雪豹。很高兴见到你~

### 教育内容
- 保护建议: 保护高山生态系统，打击非法狩猎，监测气候变化影响
- 教育内容: 雪豹是高山生态系统的旗舰物种，其生存状况反映了整个生态系统的健康状况。

## 5. 配置长江江豚示例

### 基本信息
- 动物名称: 长江江豚
- 学名: Neophocaena asiaeorientalis
- 动物类型: YangtzeFinlessPorpoise

### 生物学信息
- 栖息地: 长江中下游干流及洞庭湖、鄱阳湖
- 食物: 小鱼、虾类
- 保护状态: 极危 (Critically Endangered)
- 趣味知识: 长江江豚有标志性的微笑嘴型，被称为'微笑天使'

### AR显示设置
- 默认缩放: 0.7
- 生成偏移: (0, 0.2, 0)
- 识别距离: 2.5米

### AI对话设置
- 性格描述: 说话活泼可爱，喜欢用'咕噜咕噜'等拟声词，喜欢在长江里游泳
- 欢迎消息: 咕噜咕噜~我是嘟嘟，长江里的微笑天使江豚！

### 教育内容
- 保护建议: 减少水污染，控制航运交通，保护长江生态环境
- 教育内容: 长江江豚是长江生态系统的旗舰物种，保护江豚就是保护长江生态。
