# 3D模型资源

此文件夹包含所有濒危动物的3D模型资源。

## 模型规范

- 格式: FBX推荐，OBJ也可接受
- 大小: 尽量优化以适应移动设备
- 动画: 包含基本动作（行走、觅食、休息等）
- 贴图: PBR材质，压缩格式以减少内存占用
